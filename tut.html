BASIC HTML TAGS

1) !DOCTYPE => Defines the document type

2) "head tag" => Contains metadata/information for a document

3) "body tag" => Defines the documents

4) "h1 tag to h6 tag" => These are the HTML headings. The sizes of the heading differs from h1-h6 (decreasing from h1).

5) "p tag" => This tag is used to write down a paragraph. WHile writing a paragraph you can use this tag.
